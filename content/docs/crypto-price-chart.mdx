## Features

- Interactive **time range tabs** (1H, 1D, 1W, 1M, 1Y, ALL)
- Smooth **animated tab transitions**
- **Hover price tracking**
- Dynamic **percentage change indicator**
- Gradient **area chart visualization**
- Displays **low, high, and holdings value**
